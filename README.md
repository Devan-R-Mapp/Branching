## Git Command Cheat Sheet
Includes Basic and Information commands.
### Basic Commands

* 'git init'
* 'git status'

* 'git add .'
* 'git commit -m "message"'

### Information Commands
* 'git status'
* 'git log'
* 'git log --oneline'
* 'git '

### Branching commands
* 'git branch'
* 'git branch newBranch'
